# V3_Node

vue3+axios+pinia+Mysql+node 的一套完整独立前后端分离项目，在线演示地址http://107.151.249.219:9999

前端启动命令 npm run serve
后端启动命令 npm start
默认账号密码 admin admin

## Project setup

```
cnpm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
